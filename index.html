    What this is

    A World Cup sweep. Everyone playing is randomly dealt a share of the 48 teams. As teams get knocked out, players drop away, and whoever owns the eventual champion wins. Last team standing.

    You run it for free on your own GitHub page. One person is the admin (runs the draw, can fix scores). Everyone else just opens the link and watches it update live.

    Set it up · one time, about 10 minutes

    1. Make your own copy. On the template repo page on GitHub, click the green Use this template button → Create a new repository. Give it a name (anything, e.g. office-world-cup), set it to Public, and create it.
    2. Turn on the website. In your new repo: Settings → Pages. Under "Source" pick Deploy from a branch, choose branch main and folder / (root), then Save. After a minute your site is live at your-username.github.io/your-repo-name.
    3. Become the admin. You need a token so the app can save your draw and results for everyone.
      • On GitHub: your avatar (top right) → Settings → Developer settings (bottom of the left menu) → Personal access tokens → Fine-grained tokens → Generate new token.
      • Repository access: "Only select repositories" → pick your sweep repo.
      • Permissions: Repository permissions → Contents → Read and write.
      • Generate it and copy the token.
      • Open your site → Settings tab → paste it into "Personal access token" → Save config. The badge near the top should read Admin · synced.
      The token lives only in your browser and is never shared. Keep it private, like a password.
    4. Name your sweep. Settings → Group Branding: set the group name, a tagline, and what to call the people in it (player, lad, member, whatever). Save branding.

    Run the draw

    1. Go to The Draw tab and add everyone's name.
    2. Draw the order — shuffles who gets picked when.
    3. Draw the teams — deals the 48 teams out. Pick a draw style (Pots by ranking is the fairest spread). Any teams left over go to House (the pot).
    4. Happy with it? Hit Lock the draw so it can't be changed by accident.

    During the tournament

    Results update on their own, roughly every 20 minutes while games are on. Match Centre, Groups, Knockout and Standings all fill in automatically. Nothing for you to do.

    If a score is ever wrong or slow, the admin can set it by hand on the Teams tab (mark a team out, revive it, or crown the champion). The auto-update will reconcile on its next run.

    Share it

    Send everyone your site link (your-username.github.io/your-repo-name). They see the whole thing live, read-only. Only you, with the token saved in your browser, can change anything.

    If something's not right

    • No "Admin" badge / it won't save: your token is missing, or doesn't have Contents: Read & write, or is pointed at the wrong repo. Redo step 3.
    • Results aren't updating: check the Actions tab on your repo is enabled. Auto-updates only run during the tournament dates; to force one, go to Actions → "Update results" → Run workflow.
    • Wrong or old data showing: hard refresh the page (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows).
